# Executive Summary

SafeCloud is a secure cloud storage application developed as a final year project. The application allows users to register, log in, upload files, organize files into folders, and manage stored data.

The objective of this assessment was to identify common web application security risks in the application and evaluate whether the platform follows secure design principles.

The assessment focused on authentication, authorization, file upload handling, session management, input validation, and access control.

This report demonstrates a practical security assessment approach using OWASP Top 10 methodology and manual testing techniques.
