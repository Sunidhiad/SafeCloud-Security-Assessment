# SafeCloud Security Assessment

This repository contains a security assessment of SafeCloud, a secure cloud storage web application built with Next.js, Supabase, authentication, file upload, folder management, and access control features.

## Assessment Focus

- Authentication Testing
- Authorization Testing
- File Upload Security
- Input Validation
- Session Management
- Access Control
- Basic OWASP Top 10 Mapping

## Tools Used

- Burp Suite
- Browser Developer Tools
- Manual Testing
- OWASP Top 10 Methodology
